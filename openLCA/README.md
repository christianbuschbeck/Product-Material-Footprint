1. open openLCA

2. Click file>import>file and browse to the "create_PMF_Method_viacf.py" file.

3. Click on the "Scripts" folder in the openLCA navigation and open the "create_PMF_Method_viacf.py" script

4. Under "CHARACTERIZATION FACTORS" the directory to the folder with characterization factors must be set. To do this with a Mac, you can use the finder to browse to the target folder, go to the path on the bottom of the finder screen, right click the target folder and click "copy as pathname". Then paste between the parenthesis in cf_dir = "".

5. Run the script by clicking on the green "play" sign in the topleft corner of the openLCA window.
